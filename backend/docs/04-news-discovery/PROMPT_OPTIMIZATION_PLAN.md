#### 1. `publish_news_post(news_post: NewsPost) -> NewsPost`

–ü—É–±–ª–∏–∫—É–µ—Ç –æ–¥–Ω—É –Ω–æ–≤–æ—Å—Ç—å:
1. –ü—Ä–æ–≤–µ—Ä—è–µ—Ç —Å—Ç–∞—Ç—É—Å (`draft`)
2. –ü–µ—Ä–µ–≤–æ–¥–∏—Ç –Ω–∞ en, de, pt —á–µ—Ä–µ–∑ `TranslationService.translate_news()`
3. –°–æ—Ö—Ä–∞–Ω—è–µ—Ç –ø–µ—Ä–µ–≤–æ–¥—ã –≤ –ø–æ–ª—è `title_{lang}`, `body_{lang}`
4. –ú–µ–Ω—è–µ—Ç —Å—Ç–∞—Ç—É—Å –Ω–∞ `published`

```python
def publish_news_post(news_post: NewsPost) -> NewsPost:
    """
    –ü—É–±–ª–∏–∫—É–µ—Ç –Ω–æ–≤–æ—Å—Ç—å: –ø–µ—Ä–µ–≤–æ–¥–∏—Ç –Ω–∞ –≤—Å–µ —è–∑—ã–∫–∏ –∏ –º–µ–Ω—è–µ—Ç —Å—Ç–∞—Ç—É—Å –Ω–∞ 'published'.
    
    –ü–æ—Å–ª–µ –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏–∏ –ø—Ä–æ–º–ø—Ç–æ–≤ –Ω–æ–≤–æ—Å—Ç–∏ —Å–æ–∑–¥–∞—é—Ç—Å—è —Ç–æ–ª—å–∫–æ —Å —Ä—É—Å—Å–∫–∏–º —Ç–µ–∫—Å—Ç–æ–º.
    –≠—Ç–∞ —Ñ—É–Ω–∫—Ü–∏—è –¥–æ–±–∞–≤–ª—è–µ—Ç –ø–µ—Ä–µ–≤–æ–¥—ã –Ω–∞ –∞–Ω–≥–ª–∏–π—Å–∫–∏–π, –Ω–µ–º–µ—Ü–∫–∏–π –∏ –ø–æ—Ä—Ç—É–≥–∞–ª—å—Å–∫–∏–π
    –ø–µ—Ä–µ–¥ –ø—É–±–ª–∏–∫–∞—Ü–∏–µ–π.
    """
    if news_post.status != 'draft':
        raise ValueError(f"News post {news_post.id} is not a draft")
    
    translation_service = TranslationService()
    
    translations = translation_service.translate_news(
        title=news_post.title,
        body=news_post.body,
        source_lang='ru',
        target_languages=['en', 'de', 'pt']
    )
    
    # –°–æ—Ö—Ä–∞–Ω—è–µ–º –ø–µ—Ä–µ–≤–æ–¥—ã
    for lang, trans in translations.items():
        if trans and trans.get('title') and trans.get('body'):
            setattr(news_post, f'title_{lang}', trans['title'])
            setattr(news_post, f'body_{lang}', trans['body'])
    
    news_post.status = 'published'
    news_post.save()
    
    return news_post
```
